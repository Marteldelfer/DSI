O projeto consiste no aplicativo mobile multiplataforma **Filmeia** — um app de **avaliação e recomendação de filmes** para Android e iOS.

## ✨ Funcionalidades

- 🔍 Busca e descoberta de filmes
- ⭐ Avaliação de filmes pelos usuários
- 🎯 Recomendações personalizadas
- 📱 Interface responsiva para Android e iOS

## 🚀 Como Executar

### Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) (versão LTS recomendada)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/client) no seu dispositivo físico **ou** um emulador configurado

### Passo a passo

**1. Clone o repositório**

```bash
git clone https://github.com/Marteldelfer/DSI.git
cd DSI
```

**2. Acesse o diretório do aplicativo**

```bash
cd aplicativo
```

**3. Instale as dependências**

```bash
npm install
```

**4. Inicie o servidor de desenvolvimento**

```bash
npx expo start
```

### Executando no dispositivo / emulador

Após iniciar o servidor, escolha uma das opções:

- 📱 **Dispositivo físico:** Escaneie o QR Code com o app [Expo Go](https://expo.dev/client)
- 🤖 **Android:** Siga o guia de [configuração do emulador Android](https://docs.expo.dev/workflow/android-studio-emulator/)
- 🍎 **iOS:** Siga o guia de [configuração do simulador iOS](https://docs.expo.dev/workflow/ios-simulator/) *(requer macOS)*
